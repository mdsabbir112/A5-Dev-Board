Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM using JavaScript. The most common methods are getElementById() to select an element by its unique ID, getElementsByClassName() to select elements with a specific class, and getElementsByTagName() to select all elements with a given tag name. Modern approaches include querySelector() which selects the first element that matches a CSS selector, and querySelectorAll() which selects all matching elements as a NodeList. Among these, querySelector() and querySelectorAll() are the most flexible and widely used because they support complex CSS-style selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference between innerHTML, innerText, and textContent lies in how they handle text and HTML within elements. innerHTML returns or sets the HTML content inside an element, meaning it can include and render HTML tags. innerText returns or sets the visible text inside an element, taking CSS styles (like display: none) into account, and it triggers a reflow to compute the visible content. textContent returns or sets all the text inside an element exactly as it appears in the DOM, without considering styles or rendering, and it doesnâ€™t interpret HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where instead of adding event listeners to multiple child elements, you add a single event listener to their parent element. This works because of event bubbling, where an event triggered on a child element automatically bubbles up to its parent. By checking the event.target, you can determine which child triggered the event. This approach improves performance, reduces memory usage, and makes it easier to handle dynamically added elements. For example, attaching one click listener to a instead of every inside it is event delegation in action.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process where an event starts from the innermost element (the one that was actually clicked or interacted with) and then moves upward through its parent elements, one by one, until it reaches the root (document). For example, if you click on a button inside a div, the click event first runs on the button, then on the div, and finally on the document (if listeners exist). Event bubbling allows techniques like event delegation, where you handle events for multiple child elements using a single listener on their parent.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create, add, and remove elements in JavaScript using DOM manipulation methods: Create: Use document.createElement('tagName') to create a new element. Example: let newDiv = document.createElement('div'); Add: Use appendChild() or append() to add the new element to the DOM. Example: document.body.appendChild(newDiv); Remove: Use removeChild() or the newer remove() method to delete an element. Example: newDiv.remove(); or parentElement.removeChild(childElement); These methods let you dynamically control content and structure in a webpage.
